O que são Machine Learning?

O machine learning (ML) é o subconjunto da inteligência artificial (IA) que se concentra na construção de sistemas que aprendem, ou melhoram o desempenho, com base nos dados que consomem.

Modelo Supervisionado:

É definido pelo uso de conjuntos de dados rotulados para treinar algoritmos que classificam dados ou preveem resultados com precisão.

Modelo Não Supervisionado:

Primeiro, usamos um algoritmo e dados de exemplo para treinar um modelo. Em seguida, integramos o modelo à aplicação para gerar inferências

em tempo real e em escala.

Modelo Semi-Supervisionado:

Trata os dados rotulados da mesma forma que o aprendizado supervisionado. Ele faz previsões e calcula pesos para diferentes variáveis. O aprendizado

de máquina semi-supervisionado usa pontos de dados não rotulados para tornar o modelo mais consistente.
